# ExtractWarPackage README

Extract Java War package to the current path.

## Features

Extract Java War package to the current path.

## Requirements

vscode: `^1.30.0`

## Usage

    1.Press F1 or Ctrl+Shift+P to show all commands.<br>
    2.Use command `Extract War Package`.
    3.Enter the name of the war package.

## Extension Settings

`null`
